# G-1010: Try to label your sub blocks.

!!! tip "Minor"
    Maintainability

## Reason

It's a good alternative for comments to indicate the start and end of a named processing.

## Example (bad)

```
begin
   begin 
      null;
   end;

   begin
      null;
   end;
end;
/
```

## Example (good)

```
begin
   <<prepare_data>>
   begin 
      null;
   end prepare_data;

   <<process_data>>
   begin
      null;
   end process_data;
end good;
/
```
